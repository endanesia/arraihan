# Folder Foto Mutawwif & Tour Leader

Folder ini digunakan untuk menyimpan foto-foto mutawwif dan tour leader yang ditampilkan di homepage.

## Spesifikasi Foto

- **Format**: JPG, PNG, GIF, atau WEBP
- **Ukuran**: Maksimal 2MB per file
- **Rasio**: Sebaiknya 3:4 (portrait) untuk tampilan optimal
- **Background**: Menggunakan background merah sesuai dengan desain (seperti pada contoh)
- **Resolusi**: Minimal 600x800px untuk kualitas terbaik

## Cara Upload

1. Login ke admin panel
2. Masuk ke menu **Homepage â†’ Mutawwif & Tour Leader**
3. Klik tombol **Tambah Mutawwif**
4. Isi nama dan jabatan
5. Upload foto dengan klik tombol **Pilih File**
6. Simpan

## Catatan

- Foto akan otomatis di-resize untuk tampilan yang konsisten
- Gunakan foto dengan latar belakang merah untuk konsistensi visual
- Nama file akan otomatis diganti dengan format: `mutawwif_YYYYMMDD_HHMMSS_xxxxx.ext`
